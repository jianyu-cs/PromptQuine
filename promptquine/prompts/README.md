# Prompt Collection Guideline
We provide the ICL prompts here we used in our experiments. 

Note that we also include LLMLingua & LLMLingua2 compressed prompts in some of the tasks. 

* ```classification_prompts```: Classification and multi-choice QA ICL prompts.
* ```sentiment_transfer_prompts```: Sentiment transfer ICL prompts.
* ```jailbreaking_prompts```: Jailbreaking ICL prompts.
* ```reasoning_prompts```: Chain-of-thought reasoning ICL prompts.

* ```others```: Random sentences and orthogonal instructions from Pile and NI, specifically from [Prompt-Waywardness](https://aclanthology.org/2022.naacl-main.266/).
